Rozfazovany postup:

1. nakreslit objekt - operace na stane klienta - javascript x canvas ...
2. zachytit souradnice lomovych bodu objetu pri pohybu mysi - na strane klienta - JS
3. zachycene souradnice prevest do podoby prosteho textu a udelat z ni promenou - na strane klienta - JS + docasna pamet X PHP a docasny soubor, kam si je zapise a na klientovi ulozi
4. preformatovani textoveho retezce - JS x PHP dle volby v kroku 3
5. pripojeni na sde (pprostorrovou databazi) - server - PHP
6. pomoci SQL sekvence zapsat do db o predem zname strukture - server - PHP
